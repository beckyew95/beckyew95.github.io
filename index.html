HTML and CSS

HTML

Html stands for Hypertext Markup Language, it is responsible for the structure of a websites content.


Semantic HTML

Semantic elements provide information about the content between the opening and closing tags. This helps tell other developers, search engines and devices for people with disabilities what the content is, which in turn helps them read the page more clearly.
Examples of non-semantic tags are <div> and </span>.

Note: how to show html code characters as text.(Look up htmlspecialchars)

Some examples are:
  • & (Ampersand) typed as &-a-m-p
  • " (Double Quotes) typed as &-q-u-o-t (when ENT_NOQUOTES is not set)
  • < (Less Than) typed as &-l-t
  • > (Greater Than) typed as &-g-t

Adding Visuals to HTML

Images

Use <img src="url"> to add an image to an html file. This is a self closing tag.
Add alt="info about image", this helpf visually impaired web users to know what the image is of, but will not be visible as text on the browser.
Under the src to add a local file, use "./" to navigate through project folder and "../" to back a folder.

Audio

Use <audio> to add an audio file, this is not a self closing tag. Can add the source file using <source src="url" type="audio/mp4">.
Use optional attributes in the opening audio tag:

  • autoplay = play when webpage opens
  • controls = to add control buttons to audio

Video

Use <video> to add a video file, this is not a self closing element. Text added in between opening and closing tag will show if the video is unable to load, recommend adding "video not supported".
Use optional attributes in the opening audio tag:

  • autoplay = play when webpage opens.
  • controls = to add control buttons to video.
  • loop = to play video on a continuous loop.

Gifs

Use <embed src="url"> to add a gif to an html file, this is the old method for adding all audio/video files, however above specific methods are now preferred for those files.

CSS

CSS stands for Cascading Style Sheet, it is responsible for the styling of a websites content. Link a stylesheet to an html file within the <head> of the html file using:
<link href="style.css" ref="stylesheet"> - this is a self closing tag.
Separating selectors with a space will mean the styling applies to content under both the selectors written, separating selectors with a comma will apply the styling to selectors with either selector.
Styling will be applied to selectors based on specificity, with the styling within the most specific selector being used, unless "!important" is used to show an important selector as this will override all other styling.
Adding an asterix as the first selector on the stylesheet will add styling for all, set margin:0 and padding:0 to override any defaults set by the user agent (browser).


Pseudo-Class Selectors

Pseudo-class selectors are keywords that start with a colon (:) and are used to select elements in a specific state.

SelectorFunction
:focusA link that is focused (e.g., moved to by a keyboard user using the Tab key or something similar, or programmatically focused using HTMLElement.focus()
:visitedA link that has already been visited (exists in the browser's history).
:disabledA link that is no longer active.
:activeA link that is activated (e.g., clicked on).
:hoverA link that is hovered over by a user's mouse pointer.

Box Model

All elements on a web page are interpreted by the browser as being in a box. Aspects of this box can be altered using:

  • dimensions; width or height
  • border
  • padding
  • margin; note: horizontal margins will add, but vertical margins will default to the highest value present only.
  • overflow (what happens when the content exceeds the size of the box)


Using margin: 0 auto; this will center an element only if there is a width set for that element.

Default setting for box-sizing is content box ==> overall box dimensions = height + border + padding.
Change to box-sizing=border-box ==> overall box dimension = height
Link to helpful MDN Page on Box Sizing

overflow=hidden; will hide any overflowing content.
overflow=scroll; will add a scrollbar to the box.
overflow=visible; will display overflow content outside of the box, this is the default setting.

Positioning HTML content using CSS

    The position of the html box can be altered using the below selectors:

  • position;
    • static (default setting)
    • relative; position relative to parent element, can set top: 50px etc.
    • absolute
    • fixed; pinned to a specific spot on the page, similar to freezepane in excel
    • sticky
  • display;
    • block (take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted),
    • inline (take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted),
    • inline-block (elements can appear next to each other, and can have their width and height manually adjusted).
  • z-index; sends to front or back to show how elements will stack when overlapping, numberical value.
  • float; only works for relative and static positions.
  • clear; defines how elements should behave when they bump into eachother.(left, right, both, none)
